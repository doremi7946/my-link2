# 마이링크 (MyLink) - 화면 와이어프레임 (Wireframe)

본 와이어프레임은 기존의 **네오브루탈리즘(Neobrutalism)** 디자인 요소(두꺼운 테두리, 강한 그림자 등)를 반영할 수 있도록 **모바일 기기(Mobile-First)**를 기준으로 설계되었습니다.

## 1. 퍼블릭 화면 (일반 방문자 접속 시)

```text
+-------------------------------------------------+
|                                                 |
|                                   [공유 🔗]     |
|                                                 |
|               ( 프로필 이미지 )                 |
|                                                 |
|             @username (displayName)             |
|                                                 |
|                                                 |
|  +-------------------------------------------+  |
|  |  [🌐] 나의 포트폴리오 웹사이트 바로가기   |  |
|  +-------------------------------------------+  |
|                                                 |
|  +-------------------------------------------+  |
|  |  [🐦] 트위터 (X) 채널 바로가기            |  |
|  +-------------------------------------------+  |
|                                                 |
|  +-------------------------------------------+  |
|  |  [📷] 인스타그램 구경하기                 |  |
|  +-------------------------------------------+  |
|                                                 |
|                                                 |
|                [마이링크 로고]                  |
|                                          🔒     |
+-------------------------------------------------+
```
> **참고:** 우측 하단 구석의 🔒 (자물쇠 아이콘)은 눈에 잘 띄지 않게 작게 배치되며, 소유자가 편집 모드로 진입하기 위한 '구글 웹 로그인' 트리거로 작동합니다.

---

## 2. 퍼블릭 화면 (빈 화면 상태 - Empty State)

```text
+-------------------------------------------------+
|                                                 |
|                                   [공유 🔗]     |
|                                                 |
|               ( 프로필 이미지 )                 |
|                                                 |
|             @username (displayName)             |
|                                                 |
|                                                 |
|                                                 |
|         아직 등록된 링크가 없습니다.            |
|                                                 |
|                                                 |
|                                                 |
|                [마이링크 로고]                  |
|                                          🔒     |
+-------------------------------------------------+
```
> **참고:** 링크가 하나도 없을 때는 중앙에 깔끔한 기본 안내 문구 텍스트만 노출됩니다.

---

## 3. 에디터 화면 (소유자 로그인 시 - 인라인 편집 모드)

```text
+-------------------------------------------------+
|                            [🚪 로그아웃]        |
|                                                 |
|               ( 프로필 이미지 )                 |
|                                                 |
|             @username (displayName)             |
|                                                 |
|                                                 |
|     --- [ 인라인 텍스트 수정 모드 진입 시 ] --- |
|  +-------------------------------------------+  |
|  | 명칭: [나의 포트폴리오 바로가기       ]   |  |
|  | URL:  [https://my-portfolio.com       ]   |  |
|  |                      [ ✖ 취소 ] [ ✅ 저장]|  |
|  +-------------------------------------------+  |
|                                                 |
|     --- [ 일반 뷰어 모드 (수정/삭제 가능) ] --- |
|  +-------------------------------------------+  |
|  | [🐦] 트위터 (X) 채널 바로가기     [✏️][🗑️]|  |
|  +-------------------------------------------+  |
|                                                 |
|  +-------------------------------------------+  |
|  |            [ + 새 링크 추가 ]             |  |
|  +-------------------------------------------+  |
|                                                 |
|     [ ✅ 저장되었습니다! ] (토스트 알림 팝업)   |
+-------------------------------------------------+
```
> **참고:** 소유자가 로그인하면 숨겨져있던 [수정✏️], [삭제🗑️], [+ 새 링크 추가] 버튼이 등장합니다. 
> 수정을 누르면 즉석에서 해당 버튼 틀을 유지한 채 텍스트를 고칠 수 있는 인라인 폼(Input) 형태로 변경됩니다.
> 저장 혹은 삭제 시 화면 하단에 `Toast(토스트)` 메시지로 결과 피드백을 전달합니다.
